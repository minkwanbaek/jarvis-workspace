# Master Pack: samsungsds-enterprise-light

## Purpose
Body-slide master inspired by recent Samsung SDS presentation materials.

## Core layout
- aspect ratio: 16:9
- background: white or near-white
- title: left aligned, upper-left zone
- content: structured grid, usually 2-column or card layout
- footer: minimal, optional page number / meta

## Palette
- primary navy: #003366
- secondary blue: #00A3E0
- text dark: #2E2E2E
- text muted: #6B6B6B
- divider: #EAEAEA
- background: #FFFFFF

## Typography
- Korean: Noto Sans KR-class sans serif
- English: Inter-class sans serif
- H1: 40-48pt bold
- H2: 28-32pt semibold
- body: 16-18pt regular
- caption: 12-14pt muted

## Component rules
- use thin gray dividers
- prefer cards, tables, numbered sections, and phase boxes
- avoid heavy shadows and decorative gradients
- keep charts/tables clean and businesslike

## Suitable slides
- agenda / executive summary
- current issues
- solution overview
- expected effects
- roadmap / phases
- closing summary
