# Samsung SDS component notes

## Reusable blocks
- title + summary lead
- 2x2 issue cards
- numbered goal list
- 4-card direction block
- 3-phase roadmap cards
- thin-line data table

## Alignment rules
- left-aligned titles
- visual center slightly above mathematical center
- stable outer margins
- cards align to a consistent baseline

## Use rule
- cover uses `samsungsds-cover-dark`
- body uses `samsungsds-enterprise-light`
- avoid carrying cinematic dark styling deep into body slides
